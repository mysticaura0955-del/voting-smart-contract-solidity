# 🗳️ Voting Smart Contract

A decentralized voting smart contract written in Solidity.

## Features

- Owner can add candidates
- Duplicate candidate prevention
- One wallet = One vote
- Vote counting
- Winner calculation
- Candidate existence validation
- Owner access control using modifiers

## Technologies

- Solidity ^0.8.20
- Remix IDE

## Smart Contract Concepts Used

- Structs
- Mappings
- Modifiers
- Constructors
- Access Control
- Loops
- Require Statements
- Storage Variables

## Future Improvements

- Automatic Candidate IDs
- Voting Deadline
- Event Logging
- Candidate List Function
